= WildFly 16 Upgrade

Passport server was upgraded to use WildFly 16 under the covers.

== SmallRye Health and Metrics extensions

Passport now comes enabled with the SmallRye Health and Metrics extensions which provides standard health and metrics
endpoints. We will add some documentation as well as Passport specific metrics soon.

= PS256 support

Thanks to https://github.com/tnorimat[tnorimat] Passport now has support for signing and verifying tokens with PS256.

= MP-JWT Client Scope

New built-in client scope to make it easy to issue tokens following the Eclipse MicroProfile specification.
